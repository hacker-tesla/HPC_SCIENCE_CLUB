Proximity Sensor

Proximity Sensor is the most common sensor that everyone use in mobile phone. Basically it can detect any obstacle.
Basically we answar two question What is this? and How it works? Hope you got your first answar is previous text.

Next question is How proximity sensor works?
There a Transmiter[ Tx ] and a Reciver[ Rx ]. When Reciver received the single from transmiter it transfer it to the LM358 IC. IC process the signal and turn the LED on or off. There is also a 10K Variale Resistor by which we can increase or dicrease the transmitter signal range. For this project we need an LM358 IC, some resistor and transmiter and receiver.
