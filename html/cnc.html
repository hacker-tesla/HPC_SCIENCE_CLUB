CNC Machine

CNC Machine is a machine that can write any type of text, draw any type of picture. Even it can copy any type of Signiture.

CNC means Computer Numeric Control. Now the question is how cnc machine works?
First we need to convert text or picture into G-CODE. Then we load this code in circuite. Curcuite control two stepper motor and a servo motor. As a result we get a picture text like human draw. Two steeper works in X and Y axis and servo motor works in Z axis.
