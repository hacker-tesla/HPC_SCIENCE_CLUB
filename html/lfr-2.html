Line Follower Robot

Line Follower Robot in short LFR. LFR can follow your line and reach the dead line perfectly. There are two types of line that is White Background Black Line and
Black Background White Line.
Basically we answar two question What is this? and How it works? Hope you got your first answar is previous text.

Next question is How LFR works?
We make LFR is so many way like only using circuit, using micro-controlar, using Arduino. We make LFR in third way (Arduino). For this project we need PCB [Printed Circuite Board] board and so many proximity secsor. This sensor detect the black or white line. It help the robot to follow the line. It like eye for LFR.
